# InHouse-Service
In House Services will be the web application that provides functionality described in the product functions section. It includes the Admin, Professionals/Employee and Customers section and their different functionalities. In addition, the application will have the very user- friendly user interface so user can easily access all functionalities.  Covid-19 Pandemic badly effects on the income of some workers and it difficult to find them work and also for customers to find the workers. So, keeping in mind all this situation In House Services will help customers to find the workers to do their work and the professional workers to find the source of income.
